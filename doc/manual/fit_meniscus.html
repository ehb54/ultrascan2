Meniscus Fitter:

You can start the UltraScan Meniscus Fitter module from the Utilities menu by clicking on "Meniscus Fitter".

Operation:

This module is intended to fit the residual mean square deviations (RMSD) produced by a 2-dimensional spectrum fit with multiple meniscus position fits to a polynomial to find the optimal meniscus position. To start, copy the meniscus positions and the associated RMSD values into the editor field of the module. Next, click on the "Plot" button to create a fit. The values will be plotted in the graph window. The default order of the polynomial is set to 2nd order (a parabola, the first-order polynomial is the function of a straight line), and the fit will be plotted as well and the minimum position identified both in the plot window and also in the "Minimum" text field in the module. When fitting, the coefficients of the polynomial fit will be shown. The "Reset" function will reset all fields. If no data is listed in the editor, pressing the "Plot" button doesn't have any effect.

PLEASE NOTE:

The data format requires that the data is entered as a space separated array, with meniscus positions entered first, and RMSD values second, as shown in the example above. You can copy and paste these values directly from the e-mail message sent by the supercomputer, and edit the values using common editor functions.

Initialize Database Template:

This function allows you to initialize a new database with the appropriate table structure. If the selected database already exists, it will be overwritten with an empty version. Use with caution!

Explanation of fields and buttons:

  • To initialize a database, enter the name of the database in the field labeled "Initialize DB Template:". Click on "Initialize" to initialize the database with a new table structure.

    CAUTION!

    This function should only be used by the administrator. Accidental use of this function will delete all database contents without the possibility of recovery.

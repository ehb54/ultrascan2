3-Dimensional Solute Display and Residual Plot Control:

This program allows you to control the display of the 3-dimensional solute plots and the Residual plots. You can select any combination of 2 of the following parameters to display a 3-dimensional plot of your model file:

  • Molecular Weight
  • Sedimentation Coefficient
  • Diffusion Coefficient
  • Frictional Coefficient
  • Frictional Ratio

Clickin on "Residuals Plot" allows you to save the time and radial invariant noise contribution vectors to file, and allow you to manipulate the residuals plots and display various residuals associated with your experimental data.

Installation Instructions - Linux Slackware

To install the Slackware 8.1 package, simply download the tgz file from this website and install it with this command: 

installpkg ultrascan-version-686-release.tgz

If you are upgrading a previously installed Slackware package, use this syntax: 

upgradepkg ultrascan-version-686-release.tgz

UltraScan 6.0 download page

NOTE: UltraScan is distributed as a tar.gz archive. UltraScan is linked against glibc-2.3.1, the QWT presentation graphics package, and the C++ toolkit QT. The QWT libraries are now included in the UltraScan distribution and do not need to be downloaded separately. If your glibc library is incompatible with glibc-2.3.1, either upgrade or contact the author for assistance. UltraScan is available in binary form linked against the following multi-threaded QT library versions:

libqt-mt.so.3.1.2
libqt-mt.so.3.2.3
libqt-mt.so.3.3.0

Please check your system to see which library is installed on your system. On most systems, you can find out which QT version you have installed by issuing this command: 

find /usr/lib -print | grep libqt-mt

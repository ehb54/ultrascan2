UltraScan download - Debian 3.0 Linux installation notes

The standard Debian 3.0 distribution is missing some links that are required for UltraScan 7.0 and 7.1. These suggestions from Gerhard Steger (Heinrich Heine University Düsseldorf) fix the problem:

  1. libXrandr.so.2 and libXcursor.so.1 are not available under Debian 3.0 stable, only under Debian/testing. You can reload it using apt-get:
    # apt-get install libxcursor1/testing
    # apt-get install libxrandr2/testing

  2. a shared/dynamic version of libXinerama is required:
    # cd /usr/local/lib/
    # ld --whole-archive -share -o ./libXinerama.so.1 /usr/X11R6/lib/libXinerama.a
    # ln -s libXinerama.so.1 libXinerama.so
    # ldconfig
